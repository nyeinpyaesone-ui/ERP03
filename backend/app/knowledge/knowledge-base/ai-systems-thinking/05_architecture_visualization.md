## Communication Patterns

1. **Synchronous** вҖ” Frontend calls API, waits for response (search, dashboard)
2. **Asynchronous** вҖ” Task queued, processed later (report generation, bulk import)
3. **Event-Driven** вҖ” Action triggers events, multiple subscribers react (invoice created вҶ’ notify вҶ’ audit вҶ’ analytics)
4. **Stream** вҖ” Real-time data flow (AI Copilot streaming, WebSocket updates)

## Key Insight

> The architecture is not about the AI model. It is about the **system around the model** вҖ” the routing, the context, the validation, the approval, the audit, and the feedback loops.
